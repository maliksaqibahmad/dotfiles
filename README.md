# Dotfiles Backup Repository

This repository is a backup of my configuration files for various applications that enhance my Linux desktop environment. The configurations are tailored to my preferences and include setups for the following applications:

- **i3**: A tiling window manager that allows for efficient window management and keyboard navigation.
- **Kitty**: A modern, feature-rich terminal emulator that supports GPU rendering.
- **Picom**: A compositor for X11, providing smooth window transitions and effects.
- **Rofi**: A window switcher, application launcher, and more.
- **ZSH**: A shell.

## Backup Details

The configurations are automatically backed up to this repository daily. The backup script ensures that any changes made to the configuration files are captured and pushed to the repository, maintaining an up-to-date copy of my dotfiles.
